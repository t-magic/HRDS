## 1. Jupyterのインストール

 * このyoutubeではPython3.5をインストールしていますが、最新版(3.6)をインストールしてください。

 * Jupyterの中でPython2.7も使用する場合は、[ここ](http://www.geocities.jp/penguinitis2002/computer/programming/Python/Anaconda_Python2_3.html)を参考にしてください。
